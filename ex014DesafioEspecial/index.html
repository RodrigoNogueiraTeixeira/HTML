Tags Em Html

Passe o mouse a cima das abreviações e ira aparecer o conteudo completo.

Para Adicionar um paragrafo utilize a tag p

Para Adicionar um titulo utilize h1 h2 h3..

Para Adicionar uma imagem utilize a tag img

Para Adicionar uma abreviações utilize a tag abbr

A tag code deixa o espaço quando passado em cima o mouse fica identicos a tdas letras

Para Adicionar uma lista numerada tem que ser lista ordenada, e tem que mudar o tipo "type" para numeros

Para adiocionar com Demarcadores utilize a tag ul lista nao ordenada.
